解决fedora下ifconfig命令不见了

By crackfree

Published 2016-04-02

Contents

在fedora上发现没有ifconfig、netstat等命令不见了。原因是系统是最小模式安装的，于是有了下面的解决方法：

使用以下命令，查找ifconfig属于那个包：
sudo dnf provides ifconfig
然后，安装：
sudo dnf install net-tools
