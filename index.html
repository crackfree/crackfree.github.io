hexo安装和配置

hexo简介

hexo github项目地址 是一款基于Node.js的轻量级博客框架。它只通过静态页面生成来生成博客。这使得它非常简洁，某些依赖后端的功能如留言，需要调用其他网站的服务来完成。

  • Blazing fast generating
  • Support for GitHub Flavored Markdown and most Octopress plugins
  • One-command deploy to GitHub Pages, Heroku, etc.
    • Powerful plugin system

Hexo 使用与部署

hexo部署至github

`$ hexo d  #即可将已经搭建好的博客部署到github上去`

发表文章

hexo n "你要发布的文章的题目" #如 hexo n "test",即在博客目录source/_posts/下生成test.md文件，用markdown编辑器编辑此文件即可写博客

写完博客后
hexo g #生成本地静态页面
hexo s #可在本地查看效果
